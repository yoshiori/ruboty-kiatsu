# Ruboty::Kiatsu
An ruboty handler to check pressure diff.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-kiatsu'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-kiatsu

